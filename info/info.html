About this application

You can use this webapp to plan your trip through London on one of the Santandar Bicycles available for hire on various locations in the city. Start and end locations are suggested, but you can also use the browsers geolocation feature, if available. We'll look for the closest station near you, measured in walking distance. We take into account how many persons you want to travel with - we only select stations which have enough bikes for rent at the start and enough empty slots at the end. When we have calculated your optimal route, we provide you a map as well as textual directions to get from your start point to your end point. Note that the complete application is mobile friendly!

Used technologies

The main technology used is AngularJs, for all application logic. There is a small proxy written in PHP to fetch the feed (~10 lines). Skeleton is used as the HTML5 responsive boilerplate, since it is beautifully light and easy to use. A special thanks to Font Awesome for the beautiful icons, Gregory Pike for the Angular Local Storage module, which is used for storing the station feed on the client side to improve performance, the creators of X2JS, the library which parses the XML station feed into usable JSON data, and Google for their useful API's, Google Places, Google Maps Distance Matrix and Google Maps Directions Service. Last but not least, a thanks to Transport for London, for supplying the Santandar Cycles Station Feed.
